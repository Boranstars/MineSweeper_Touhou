# MineSweeper_Touhou

![C++](https://img.shields.io/badge/language-C++20-blue.svg)
![Qt](https://img.shields.io/badge/framework-Qt6.8-green.svg)
![CMake](https://img.shields.io/badge/build-CMake-orange.svg)

一个使用了东方元素的扫雷小游戏，基于 WHUT 计算机编程强化训练大作业。

## 功能特点

- 经典扫雷玩法，融入东方风格
- 支持多种难度（简单、普通、困难、自定义）
- 月相主题皮肤（计划中）
- 排行榜和暂停功能（计划中）
